# Privacy Policy for Talking Buttons

Talking Buttons is a local-only communication board app for non-speaking
autistic children, managed by a caregiver.

## Data Storage

The app stores the following locally on the user's device:

- Button names and titles
- Recorded audio clips
- Selected images and photos
- Button colors
- Button display order
- Saved presets (button layouts)
- Language preference (Indonesian / English)
- Background settings (color or wallpaper image)

All data is stored in the device's local IndexedDB storage. No data is
uploaded to any server, cloud service, or third party.

## Data Collection and Sharing

The app does **not**:

- Collect personal information
- Create user accounts
- Upload audio recordings, images, or button data
- Use analytics, crash reporting, or tracking
- Display advertisements
- Include social features or sharing
- Sync data to the cloud
- Access the internet (beyond what Capacitor's WebView requires to serve local assets)

## Permissions

### Microphone

Used only when the caregiver records audio for a communication button.
Recordings are stored locally and never transmitted.

### Camera

Used only when the caregiver chooses to take a photo for a communication
button. Photos are stored locally as part of the button data.

### Photos / Media / Files

Used only when the caregiver chooses an existing image from the device
for a communication button. Images remain local.

### Internet

Required by Capacitor's WebView to serve the app's local HTML, CSS, and
JavaScript files. The app does not make network requests to remote servers.

## Children's Privacy

The app is designed as a caregiver-managed tool for non-speaking autistic
children. It does not collect, store, or transmit any personal information
about the child or the caregiver.

## Changes

If this privacy policy changes, the updated version will be included in
future app updates.

## Contact

For questions about this privacy policy, contact the developer through
the app's Play Store listing.
